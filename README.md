# 💪 GripMaxx - E-commerce Website

GripMaxx is a full-stack e-commerce website developed for selling gym essentials.

---

## 🚀 Features

* User authentication (login/register)
* Product listing with size and stock management
* Add to cart and order placement
* Cash on Delivery (COD) system
* Admin panel for managing products and orders
* Contact form for customer queries

---

## 🛠️ Technologies Used

* HTML, CSS, JavaScript
* PHP
* MySQL

---

## 🗄️ Database

Includes tables:

* users
* products
* orders
* order_items
* contact_messages

Database file: `gripmaxx_final.sql`

---

## ⚙️ Setup Instructions

1. Download or clone the repository
2. Set up XAMPP or any local server
3. Import `gripmaxx_final.sql` into MySQL
4. Configure database settings
5. Run project on localhost
